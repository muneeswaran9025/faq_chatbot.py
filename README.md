# FAQ Chatbot Using Python

## Description
A simple FAQ chatbot developed using Python. The chatbot answers predefined questions entered by the user.

## Features
- Answers basic FAQs
- User-friendly interaction
- Beginner-friendly Python project

## Technologies Used
- Python

## How to Run
1. Open the file in Python IDLE.
2. Run the program.
3. Enter questions such as:
   - hello
   - what is python
   - bye# faq_chatbot.py
